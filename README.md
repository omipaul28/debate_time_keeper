# Debate Timekeeper App ⏱️🎤

An efficient and customizable debate time management Android application designed for debate organizers and participants. This app automates debate timing with visual and audio cues for a seamless experience. Built using **Java** and **XML**, the app is tailored for the **Bangladeshi debating system**, with support for preset debate formats and intuitive controls.

---

## 📱 Features

- **Preset Debate Format**:
  - Includes the `3 vs 3`
  - Automatically transitions through 8 predefined stages of debate.
- **Automated Timer with Audio Cues**:
  - Visual cues with color-coded timers.
  - Bell sounds for signal points and transitions.
- **Stage Management**:
  - Timers and signals for each debater's speaking time.
  - Control buttons for starting, stopping, and resetting timers.
- **User Interface**:
  - Simple and intuitive layout for smooth navigation.
  - Visual progress indicators for each debate stage.
- **Reset Functionality**:
  - Reset the timer and app states for a new debate session.
- **About Page**:
  - Includes developer information and app usage guidelines.
